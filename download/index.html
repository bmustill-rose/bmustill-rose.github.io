System Requirements

You'll need 2 things in order to use Can You See Me:

  • A computer running Windows x
  • A webcam (both builtin and external webcams are supported)

Can You See Me has been confirmed to be compatible with NVDA, Dolphin Screen Reader and JAWS. If you're using one of these screen readers prompts will be spoken and Brailled through your screen reader. If you're not using a screen reader or one that is incompatable the tool will speak its prompts its self.
